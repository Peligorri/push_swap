# push_swap

## Description
In the push_swap project, we had to develop a program that sorts a stack of integers in ascending order.\
\
To do so, the program calculates a disorder percentage and, based on the result, chooses one of three sorting algorithms:\
\
- Simple algorithm
- Medium algorithm
- Complex algorithm
